# SchroSIM

Continuous-variable (CV) photonic quantum simulation stack for designing, compiling, validating, and executing hardware-like photonic circuits.

## Install

For package usage:

```bash
pip install schrosim
```

For source usage:

```bash
swift run schrosim-cli --help
```

## Core Capabilities

- Backend-aware CV simulation workflows (Gaussian, hybrid, Fock-aware paths)
- Compiler and IR mapping from circuit design to runtime execution
- Validation, diagnostics, and reproducibility controls for technical studies
