# core-swift

This folder contains the Swift simulation core and CLI:

- `Sources/SchroSIM` — Swift library (Gaussian/Fock/Hybrid simulators + compiler)
- `Sources/schrosim-cli` — CLI entrypoint
- `Tests/SchroSIMTests` — Swift unit tests

Build/test from repo root with:

```bash
swift build
scripts/swift-test.sh
```
